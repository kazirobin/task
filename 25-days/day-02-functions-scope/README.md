Step-by-step explanation:
1. Compare function declarations (e.g., function add() {}) with arrow functions (e.g., const add = () => {}).
2. Differentiate parameters (placeholders in function definition) from arguments (actual values passed when calling).
3. Understand global scope (accessible everywhere) vs local scope (accessible only within the function).
4. Implement basic math functions that take two numbers and return the result.
5. Combine them into a mini calculator that performs operations based on user input.
6. Add validation to ensure inputs are numbers.
