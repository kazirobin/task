**Concepts**
- Nested objects
- Destructuring
- Object methods
**Practice**

Step-by-step explanation:
1. Create objects with nested structures (e.g., { address: { city: 'X' } }).
2. Use destructuring to extract properties (e.g., const { name } = user).
3. Add methods to objects (e.g., user.addSkill = function(skill) {}).
4. Implement functions to modify the user object: push to skills array, splice to remove, reassign age.
5. Practice with the given user object.
6. Commit your code.
